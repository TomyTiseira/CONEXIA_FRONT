# Mejoras visuales en Proyectos recomendados y botones

## Cambios realizados

- Se mejoró la visualización del bloque "Proyectos recomendados para ti" para que el título y la cantidad de proyectos se muestren apilados y centrados, tanto en mobile como en desktop, logrando mayor claridad y consistencia visual.
- Se aumentó ligeramente el tamaño del icono de "Mis postulaciones" para mejor visibilidad.
- El botón de volver atrás (BackButton) ahora utiliza el color verde petróleo característico de Conexia en su borde, manteniendo el mismo grosor y estilo general.
- Se mantuvo el diseño de desktop para todos los bloques, priorizando los cambios visuales en mobile donde era necesario.

## Objetivo

- Mejorar la experiencia de usuario y la coherencia visual en la sección de proyectos recomendados y botones de acción, especialmente en dispositivos móviles.

---

**Nota:** Si se requiere ajustar tamaños, colores o alineaciones adicionales, avisar para continuar con los cambios.
