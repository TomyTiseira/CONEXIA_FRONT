# Videos para Hero Background

## Instrucción para agregar el video de fondo

Para que el video de fondo funcione en la página principal (HeroHome):

1. Coloca tu video en esta carpeta con el nombre: `hero-background.mp4`
2. El video debe estar en formato MP4 para mejor compatibilidad
3. Recomendaciones para el video:
   - Duración: 10-30 segundos (será en loop)
   - Resolución: 1920x1080 o superior
   - Tema: Tecnología, redes, conexiones, IA, digital
   - Estilo: Moderno, abstracto, profesional
   - Colores: Preferiblemente tonos azules, verdes, cyanes (acorde con la marca)

## Sugerencias de videos gratuitos

Puedes descargar videos gratuitos de:

- **Pexels Videos**: https://www.pexels.com/videos/
  - Busca: "network", "technology", "ai", "digital", "connection"
- **Pixabay**: https://pixabay.com/videos/
- **Coverr**: https://coverr.co/

## Fallback

Si no hay video disponible, automáticamente se usará la imagen de fondo ubicada en:
`/public/images/hero-background.jpg`

La imagen ya está configurada y funcionará perfectamente.
